# Flow — Emotion Interaction Firmware

This Arduino sketch powers the Flow prototype,  
a sensory device that conveys emotional connection through light, vibration, and sound.

## Hardware
- Arduino Nano / UNO
- DFRobot DFPlayer Mini
- Adafruit NeoPixel 8 LED ring
- Vibration motor module
- Dual LDR (Light sensors)

## Features
- Detects touch via light attenuation
- Plays responsive sounds (DFPlayer)
- Generates breathing LED and motor feedback
- Idle ambient sound mode after 30s inactivity
